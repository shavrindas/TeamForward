# TeamForward 프로젝트

TeamForward 프로젝트는 개발 초기 단계에서 사용될 요소의 집합으로, 다음과 같은 목적을 두고 있습니다:

- 학술적 연구 및 공유
- 기술 개발
- 효율화

## 작업 방식

프로젝트의 작업 방식은 다음과 같습니다:


상위라인
- `main`: 최종 개발 
- `release`: 하위 라인 통합 

하위라인
- `back`, `front`: 작업장 
- `dina(ai)`, `db`, `web page`: 프로젝트 라인
- `dev_내용`: 개발 라인
- `fix_내용`: 수정 라인

예시:
main
release 

back/front
project
dev_feature
fix_feature
